#### Installing Vue Plugin

Unninstall Vue past versions

    $ npm uninstall -g vue-cli
    
Install Vue

    $ npm install -g @vue/cli
    
    
#### Installing Quasar CLI

    $ yarn global add @quasar/cli
    $ yarn create quasar

or:

    $ npm i -g @quasar/cli
    $ npm init quasar
